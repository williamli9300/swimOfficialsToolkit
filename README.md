<blockquote>
  <details>
    <summary>Instructions for Installing HyTek Custom DQ Codes (<i>click to expand</i>)</summary>
    <ul>
      <li>Copy <code>dqcodesCustom.txt</code> to your <code>C:\Hy-Sport\SwMM7</code> or <code>C:\Hy-Sport\SwMM8</code> folder.</li>
      <li>Open Hy-Tek Swim Meet Manager. In the home screen, navigate to <code>Set-up</code> > <code>Meet Set-up</code>.</li>
      <li>In the bottom right of the Meet Set-Up screen, select <b>Custom DQ Codes</b> from the <b>DQ Codes</b> dropdown.</li>
      <li><i>Note: The Hy-Tek Custom DQ Codes file is sensitive to newline characters. The file must use Windows-style (<code>CRLF</code>). Please do not download the <code>*.txt</code> file from GitHub, as it is automatically converted to Unix-style <code>LF</code>. For the nerds reading this, this repository will be converted to use <code>text eol=crlf</code> at a later date.</i></li>
    </ul>
  </details>
</blockquote>
